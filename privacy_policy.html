Privacy Policy for LLM Translator & Explainer

Last updated: March 23, 2024

1. Information We Collect

Our extension collects the following types of information:

  • Selected Text: When you select text on a webpage for translation or explanation, that text is collected and transmitted to OpenAI's API.
  • API Key: Your OpenAI API key is stored locally in your browser using Chrome's storage sync feature.

2. How We Use Your Information

The collected information is used solely for:

  • Processing translations and explanations through OpenAI's API
  • Authenticating requests to OpenAI's API using your stored API key

3. Data Storage

We store:

  • Your OpenAI API key in Chrome's secure storage sync system
  • No selected text or translation results are permanently stored

4. Data Sharing

Selected text is shared with:

  • OpenAI's API for processing translations and explanations
  • Please refer to OpenAI's Privacy Policy for information about how they handle your data

5. Security

We protect your information by:

  • Using Chrome's secure storage system for API key storage
  • Using HTTPS for all API communications
  • Never storing selected text or translation results

6. Your Rights

You have the right to:

  • Remove your API key from the extension at any time
  • Uninstall the extension, which will remove all stored data

7. Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes through the extension or our website.

8. Contact Us

If you have any questions about this Privacy Policy, please contact us at giorkin@gmail.com.
